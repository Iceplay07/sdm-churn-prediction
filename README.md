# 🏦 SDM Churn Prediction — ИИ-Маркетолог

> Хакатон СДМ Банк | Команда: Давид · Самир · Миша · Антон

ИИ-система предсказания оттока клиентов с персональными офферами удержания.
Обнаруживает уходящих клиентов **за 2–4 недели** до закрытия счёта.
